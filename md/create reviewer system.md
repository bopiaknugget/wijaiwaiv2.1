# คุณคือ AI LLM engineer มีควมเชี่ยวชาญเป็นอย่างมากในการสร้าง prompt enigineering อย่างมีประสิทธิภาพ
# ความถนัด : python, streamlit
## purpose : สร้างปุ่มที่ใช้ review คุณภาพงานวิจัย

*สิ่งที่คุณต้องทำ*

Backend : 
    (1) สร้าง module ใหม่ review งานวิจัยในฐานะ อาจารย์ที่ปรึกษาที่เข้มงวด โดยให้คำแนะนำว่าส่วนไหนดีถูกต้องแล้ว อันไหนควรแก้ไขเพิ่มเติม
    (2) prompt ต้องละเอียดบ่งบอก persona อาจารย์ที่ปรึกษาที่ชัดเจน , review ข้อมูลตามแนวทางบทที่ 1-5 ของงานวิจัยให้อยู่ในรูปแบบที่ถูกต้อง
    (3) module reviewer นี้ให้แยกมาอีก 1 file เพิ่อความเข้าใจง่ายของ structure project

Frontend : 
    (1) สร้างปุ่ม ที่มี 'label adviror พร้อม icon ที่เหมาะสมสวยงาม' ใน panel ขวา ใต้ Assistant chat 
    (2) ใต้ปุ่มมี chatbox เล็ก ๆ ให้ user บอกว่าอยาก review อะไรเป็นพิเศษ ซึ่งเป็น optional ให้ user เลือกจะกรอกหรือไม่ก็ได้ 
    (3) พอกดปุ่ม app จะ ทำการดึงข้อความใน workbench text editor มาทำการ review ตาม backend logic
    (4) ผลลัพธ์จะเป็นข้อความใต้ editor ซึ่ง expand collapse ได้ (default คือ expand) 
    (5) ใน output ใช้บ่งบอก
        สีแดง  [ส่วนที่ต้องแก้]- 
        สีเขียว [ส่วนที่ดีแล้ว] 
        สีเหลือง [คำแนะนำ]
    เลียนแบบการตรวจงานของที่ปรึกษาที่เป็นคนจริง ๆ
