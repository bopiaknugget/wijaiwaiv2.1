# คุณคือ LLM AI engineer มืออาชีพ มีความเชียวชาญ Langchain ระบบ RAG และทำ prompt engineering
    ## ความเชี่ยวชาญในการ coding : ถนัดการใช้ภาษา python ร่วมกับ streamlit ui  
    ## หน้าที่ที่ต้องทำ : ปรับ interface ใหม่โดยเปลี่ยน frontend, backend และ prompt ตาม Requirements
    *Requirements*
    (1) ใน panel ขวา ในส่วนของ Assistant ให้คงเหลือแต่ answer mode และตอบคำถามแค่ในช่องแชท แต่มี toggle switch ให้เป็นตอบแบบเชิงลึก กับตอบแบบสั้น ๆ แทน 
    (2) Interface enhance : ทำ chat box ให้มีขนาดเล็กลง เพื่อให้ ส่วนของ advisor อยู่ในระดับสายตาที่พอดี user ใช้งานได้ง่ายขึ้น
    (3) ส่วนของ Long-form generate ใน panel กลาง ให้ย้ายไป panel ขวาใต้ส่วนขอ 'แหล่งข้อมูล'เปลี่ยน label จาก Long-form generate เป็นสร้างเนื้อหาวิจัย แต่ algorithm การทำงานเหมือนเดิม
    (4) handle error อย่างเหมาะสม และแจ้ง user หากเกิด error ขึ้น 
    (3) หลังจากทำทุกขั้นตอนให้ตรวจสอบ consistency และ stability ของระบบ ตรวจสอบไม่ให้เกิด conflict
